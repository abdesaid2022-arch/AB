# Nexa AI - Business Operating System

A modern, elegant, and intelligent Business OS built with Next.js 15.

## Features (MVP)

- Email + OTP Authentication
- Beautiful Dashboard
- AI Assistant
- Owner Command Center (User management + Ban)
- Automation Center
- Unified Inbox (UI)
- Sales Module

## Tech Stack

- Next.js 15
- Supabase (Auth + Database)
- Framer Motion
- Tailwind CSS
- Resend
- OpenAI / Gemini

## Getting Started

```bash
npm install
npm run dev
```

## Environment Variables

All keys are stored in `.env.local`
